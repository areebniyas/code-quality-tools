startTenantFlow() should be the first line in a try-block. endTenantFlow() should be in a finally block.

Noncompliant Code Example

void foo(){
    startTenantFlow(); // Noncompliant
    endTenantFlow(); // Noncompliant

    try {
        String myString;
        int myNum;
        startTenantFlow(); // Noncompliant
    } catch(Exception e) {
        endTenantFlow(); // Noncompliant

    } finally {

    }
}

Compliant Solution

try {
    startTenantFlow();
    int i;

} catch(Exception e) {

} finally {
    endTenantFlow();
}
